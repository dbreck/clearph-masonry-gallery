Gallery Animation Test

Testing Instructions:

  1. Load a page with the gallery shortcode
  2. Scroll down to see the gallery
  3. Images should fade in with a slight upward movement as they enter the viewport
  4. Hover over images to see the scale animation
  5. Check browser console for any JavaScript errors

Expected Behavior:

  • ✅ Images start invisible (opacity: 0)
  • ✅ Images fade in when they enter the viewport
  • ✅ Smooth upward movement during fade-in
  • ✅ Scale effect on hover
  • ✅ Better image quality for Large/Wide images
  • ✅ Responsive image sizes with srcset

Debugging:

If animations aren't working:

  1. Check if GSAP is loaded: typeof gsap !== "undefined"
  2. Check if images have "image-ready" class
  3. Check browser console for errors
  4. Verify CSS is loading properly
